Javascript学习日记：基础语法、对象、BOM、DOM、事件监听

2022-05-19

一、基础语法

如何引入

​ Javascript有两种引入方式，

第一种：直接在html文件中加入script标签

​ 像这样子

<body>

<script>
    alert("Hello Javascript");
</script>


</body>
第二种：在外部包写js文件，在html文件中导入js文件
<script src="../js/Demo.js">
    
</script>

​ Tip：导入外部js的文件中不能包含script标签

​ 且导入时html文件中的script标签不能自闭合

​ 一般会把script标签放在标签的最下面，js是脚本语言，所以为了用户的体验感受，会优先加载出页面的 基本内容，再加载动态信息，所以优先把script标签放在标签的最下面。但是如果网页的js文件是全局定义且后期不怎么修改的，也可以放在标签中。

二：基本语法

1、输出语句

window.alert("Hello Javascript");//写入警告框
document.write("Hello Javascript");//写入Html
console.log("Hello Javascript");//写入控制台

​ Tip：script标签 中语句结尾的分号可以删掉。

2、变量：

​ 2.1、js里面有var来申明变量，var变量可以存放不同类型的值。

​ 2.2、变量名的规则与java一致。

​ 2.3、ECMAscript6 新增了let关键字来定义变量，用法类似var，声明的变量只在let关键字所在的代码快生效，且不允许重复声明。

​ 2.4、ECMAscript6 新增了const关键字，用来声明一个只读的变量，一旦声明，变量的值就不能改变。

3、数据类型：

​ 3.1、原始类型

  • number
  • string
  • boolean
  • null
  • undefined

​ Tip:可以使用typeof运算符获取数据类型

4、运算符

​ 区分“”和“=”：

​ “==”：判断类型是否一致，如果不一致就进行类型转换，再对值进行比较

​ “===”：判断类型是否一致，如果不一致直接返回false，再对值进行比较

5、类型转换：

5.1、其他类型转为Number

  • string ：按照字符串的字面值，转为数字，如果字符串的字面值不是数字，则返回NaN，一般使用 parseInt

    • boolean ：true转为1，其他转为0
5.2、其他类型转为boolean
  • number:0和NaN转为false，其他转为true
  • string:空字符串转为false，其他转为true
  • null:false
  • undefined:false

6、流程控制语句

//if
var count = 3;
if (count == 3){
    alert(count);
}

//switch
var num = 3;
switch (num){
    case 1:{
        alert('今天是星期一')
        break;
    }
    case 2:{
        alert('今天是星期二')
        break;
    }case 3:{
        alert('今天是星期三')
        break;
    }case 4:{
        alert('今天是星期四')
        break;
    }case 5:{
        alert('今天是星期五')
        break;
    }case 6:{
        alert('今天是星期六')
        break;
    }case 7:{
        alert('今天是星期日')
        break;
    }
    default:alert('输入信息有误');
}

//for
var sum = 0;
for (let i = 0 ; i <= 100 ; i++){
    sum += i;
}
alert(sum);

//while
var sum = 0;
var i = 1;
while (i <= 100){
    sum += i;
    i++;
}
alert(sum);

//do...while
var sum = 0;
var i = 1;
do {
    sum += i;
    i++;
}while (i <= 100);
alert(sum);

7、函数

7.1、定义函数
function add(a,b){
    return a+b;
}
let result = add(1,2)

注意：形式参数不需要类型，返回值也不需要定义类型，可以在函数内部直接使用return即可

var add = function (a,b){
    return a+b;
}
// let result = add(1,2)
//在js中调用函数可以传入任意个参数，传进几个就使用几个，超出定义时的参数个数，超过的部分也不会干涉原本的程序运行。
let result = add(1,2,3,4)

五、DOM

​ ：能对HTML文件进行修改

5.1、Document：整个文档对象

5.2、Element：元素对象

5.3、Attribute：属性对象

5.4、Text：文本对象

5.5、Comment：注释对象

：方法
//1、getElementById:根据id属性值获取，返回一个Element对象
var img = document.getElementsByClassName("light");
alert(img);
img.src = '../images/sun.jpg';

//2、getElementByTabName:根据标签名称获取，返回Element对象数组
var divs = document.getElementsByTagName("div");
/*
    style:设置css样式
    innerHTML:设置元素内容
 */
for (let i = 0; i < divs.length; i++) {
    divs[i].style.color = 'red';
    divs[i].innerHTML = 'HH';
}

//3、getElementByName:根据name属性值获取，返回Element对象数组
var hobbies = document.getElementsByName("hobby");
for (let i = 0; i < hobbies.length; i++) {
    alert(hobbies[i]);
}

//4、getElementByClassName:根据Class属性值获取，返回Element对象数组
var clas = document.getElementsByClassName("cls");
for (let i = 0; i < clas.length; i++) {
    alert(clas[i]);
}
